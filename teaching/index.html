The "Missing Semester" Workshops

I teach several workshops on tools and techniques that most scientists and engineers are expected to pick up on their own. I call this catalogue of courses the "Missing Semester". The courses are all designed to be taught hands-on, with participants practicing commands on their own computers. Most courses below have class notes openly available for those interested to self-learn, the notes are provided under a Creative Commons (CC BY-NC-ND 4.0) license.

If you are interested in arranging one or more of these workshops at your institute, please contact me. Pricing can be discussed.

Introduction to version control with Git

 1 day workshop

This is an interactive course in which participants can learn the use of Git in the version control of code. At the end, participants will be able to initialise and manage Git repositories, make commits, use branches, and go back and solve mistakes. Teaching will be hands-on, and all participants will work on a computer to learn by experience.

Introduction to GitHub

 1 day workshop

Building on "Introduction to version control with Git", in this interactive course participants will learn how to manage their code online using GitHub. Topics include how to push and pull code to the server, how to use GitHub collaboratively, and how to deal with READMEs and licenses.

LaTeX for academics

 1 day workshop

LaTeX is a powerful type-setting tool designed for the writing of academic documents. With tools like Overleaf, LaTeX has become accessible to everyone and can greatly improve the efficiency of writing. This workshop will guide participants through the basics of LaTeX through Overleaf, and help them practice by typesetting a document of their choice.
